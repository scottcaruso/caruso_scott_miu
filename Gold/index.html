Known Issues List

Back

View Recent After Delete Card

The last item in Local Storage doesn't appear in Recent Cards after a card has been deleted. Need to refactor the code to go back to this id and/or re-key the existing items.

Edit Card Page - Slider

When editing a card, the Mana Cost slider goes to its default of "3" rather than whatever the card's default is.

Viewing Edit Card after backing out of Add Card

Going to Edit Card after using Back/Home to exit Add Card causes some elements of the form to not populate properly. This does not occur if the user Adds a card from the Add Card page.

Reset Form

The Reset Form button only clears text fields for some reason. Need to investigate further to see if we can fix it.
